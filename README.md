# Re-Con

### Contributors

- Roger Reyes
- Darius Owens

### Project scope

- Memory/concentration game that allows you to select different themes (animals, dogs, etc!), play the game, and see your stats (all-time number of wins, etc).

### Features

- 20-card memory game
- animations and turn tracker
- profile creation
- theme choice
- persistent stats page
- About Us page
